# Time-Series Database Examples

This directory will contain examples for time-series databases such as:

- **InfluxDB** - High-performance time-series database
- **TimescaleDB** - PostgreSQL extension for time-series data
- **QuestDB** - High-performance time-series database with SQL support

## Planned Examples

- Basic time-series data insertion and querying
- Downsampling and aggregation queries
- Retention policy management
- Connection pool examples

## Contributing

When adding a new time-series driver, follow the guide at:
`libs/cpp_dbc/docs/how_add_new_db_drivers.md`
